Web Based Doctor Appointment System 

- Accessible By Patient and HealthCare Center
- Patient can Login/Register themselves, List Doctor, see Doctor Details, Book Appointments, get Confirmation through mail, view Previous Visits.
- Admin can view Doctors , change Availability Status
- Future Enhancements: - Admin can manage Doctor Available Slots, Mail prescriptions
